# nano-project
## Titanic Dataset Viewer And Analysis

An exercise aimed at assimilating data from an external CSV file into a VueJs application.

## Installation

These instructions describe how to install this application on a 'clean' machine.

### Pre-installation

- Ensure [NPM] and [git] are installed on your system

### Installation

1. Setup the correct path for your installation

    ```
    cd /path/to/install/location
    ```

2. Install this code on your local system

    ```
    cd /path/to/install/location

    git clone https://github.com/ClassWare/nano-project.git
    ```

3. Install the dependencies

    ```
    npm install
    ```

4. Start the file server using

    ```
    node fs-server-csvtojson
    ```

    The file server will report that it is listening on port 4040

5. Start the application

    ```
    npm run serve
    ```

6. Your site will be available at localhost:8080
